Enunciado

Joana, a gerente de uma pequena loja, agora está lidando com um novo desafio. Ela precisa combinar os dados de estoque de duas lojas diferentes que recentemente se fundiram. Crie uma função que receba dois dicionários de estoque e retorne um novo dicionário que combine ambos, somando as quantidades de produtos para as chaves comuns. Se uma chave existir em ambos os dicionários, o valor no novo dicionário deve ser a soma dos valores das duas entradas.

Entrada: Dois dicionários com chaves comuns e valores inteiros.

Saída: Um dicionário combinado.
